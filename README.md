## 留意事項

- MOFE側に伝えなければならないこと
    - 問題作成者全員のMOFE ID(これは全員作成済みのはず)
    - ペナルティの時間
    - 順位表モード(AtCoder / ICPC)
    - コンテスト説明文(Markdown)
    - 解説URL(これは後でも)
    - 追加する問題の問題IDと配置位置
- 問題自体はコンテストページに追加後もWriter側で編集可能、コンテスト説明文は更新したら運営チームに変更してもらう必要がある
- Testerは問題作成画面から追加する
- インタラクティブは非対応
